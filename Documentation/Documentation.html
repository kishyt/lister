Structure of Css Styles

The main stylesheet is located in the / css folder ,this file is Styles.css where global styles are defined.

There are aditionals files where colors are defined . Located in the / css / skins folder.

You can choose from 6 colors : Blue , Red , Green , Purple , Orange , Pink .

You can also choose from 3 skins : Aurora, Sky , City .

If you would like to change it , open / Styles.css and edit the Skin Styles Section ( Line 27 )

Example - if you want to use the blue skin , just use : @import url ( "skins/ blue.css" ) ;

Boxed or Wide if you want to use the Boxed / Wide version , just set it on styles.css : @import url ( "skins/ boxed.css" ) ;

Note : to change the background of Boxed version please open /css/skins/boxed.css and edit line 5 .
